Algumas explicações

Para criar um componente no Angular, rode o seguinte comando:

  • ng generate component NomeDoComponente

    • Compartilhando dados do componente pai para o componente filho:

  • 1. Criar o componente filho (criamos o componente chamado "parent-component".);
  • 2. Adicionamos o "parent-component" ao HTML do componente pai (abrindo e fechando suas tags);
  • 3. No navegador, apareceu a informação de confirmação da criação desse novo componente;
  • 4. Criamos a propriedade userName = 'Rodrigo';, no TS do componente pai
  • 4. Sinalizamos no HTML do componente pai que um dado está sendo compartilhado (adicionamos [name] = 'userName';
  • 5. Criamos uma propriedade no componente filho (foi criado um Input no import do TS do componente filho);
  • 6. Adicionamos no export do componente filho, no TS, a expressão @Input() name: string = ' ';
  • 7. Utilizamos os elementos no componente. Acessamos o componente filho (parent-component) e adicionamos no HTML: Nome: name (o name é entre duas chaves).
