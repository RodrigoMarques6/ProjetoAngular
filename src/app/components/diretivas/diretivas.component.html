Diretivas

Aplicando estilos para um elemento:

  • 1. Crie um componente;
  • 2. Adicione o componente no HTML pai;
  • 3. Adicione a propriedade [ngStyle] e escreva CSS em linha para trazer estilos ao conteúdo. o CSS escrito deve estar entre aspas e entre chaves

    • Vejamos funcionando:

    Este fundo está vermelho

    Aplicando estilos a um elemento por meio de propriedades:

  • 1. Acesse o TS do componente e adicione as propriedades de estilos na parte do export (ver a parte do export, no código)

    • Vejamos funcionando:

    Recebi uma cor, um fundo dinâmico e um border dinâmico

    Aplicando estilos a um elemento por meio de classes:

  • 1. Acesse o TS do componente e adicione as propriedades de estilos na parte do export (ver a parte do export, no código). No exemplo, criamos a propriedade "classes".
  • 2. Adicione o elemento no HTML do componente filho e atribua a ele a classe especificada.
  • 3. Agora, acesse o CSS do componente filho e utilize as classes criadas no TS do mesmo componente para adicionar os estilos.

    • Vejamos funcionando:

    Recebi estilos através de classes!
